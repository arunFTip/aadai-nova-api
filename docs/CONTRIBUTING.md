# Contributing

---

# Branch Naming

```text
feature/user-management
feature/activity-logs
fix/auth-token
```

---

# Commit Convention

```text
feat:
fix:
refactor:
docs:
```

Example:

```text
feat(user-management): add pagination
```

---

# Backend Rules

- Use FormRequest validation
- Use single-action controllers
- Keep business logic out of controllers
- Follow modular structure
- Reuse response helpers

---

# Pull Request Checklist

- [ ] API tested
- [ ] Validation handled
- [ ] ACL checked
- [ ] Error handling added
- [ ] Activity logs considered
